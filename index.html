1. Descripción del emprendimiento

MALEJA Calzado

¿En qué consiste?

Emprendimiento familiar de calzado femenino que vende principalmente por WhatsApp e Instagram. Problema identificado: catálogos desordenados, respuestas lentas y falta de presencia web profesional que genera pérdida de ventas.

Producto estrella: Plataforma web de catálogo integrada con WhatsApp Business para convertir navegación en contactos directos con mensajes prellenados.

Tarjeta de presentación real con códigos QR funcionales

2. Diseño y descripción del producto estrella

Diseño responsive adaptable a móvil y desktop

Descripción técnica y funcional

¿Qué hace?

  • Catálogo web responsive con filtros, búsqueda y paginación
  • Ficha de producto con galería y botón WhatsApp directo
  • Panel administrativo con CRUD para autogestión
  • Mensajes WhatsApp prellenados por producto

¿Qué lo hace diferente?

  • Enfoque en microempresas locales (Cali)
  • Conversión directa a WhatsApp (no e-commerce tradicional)
  • Velocidad móvil optimizada
  • Capacitación y soporte incluido

3. Implementación técnica (Competencias ADSO)

Stack tecnológico

PHP 8 MySQL JavaScript CSS Grid SSL
  • Backend: PHP 8 + MySQL con PDO preparado
  • Frontend: JavaScript vanilla + CSS responsive
  • Seguridad: password_hash, variables .env, sanitización
  • Performance: Lazy loading, imágenes ≤200KB

Arquitectura del sistema

  • Separación de responsabilidades (MVC adaptado)
  • Panel administrativo con autenticación segura
  • Diseño mobile-first responsive
  • SEO básico y accesibilidad (alt, aria-label)
  • Hosting en Hostinger con dominio propio
